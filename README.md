## Overview 🗺️

This is a student-led **ALGOSUP** project aimed at creating a **REST API** to determine the quickest path between two points within a dataset containing **23,947,347 nodes**. The implementation is built using **C++**, leveraging its performance capabilities for efficient processing.
The **REST API** must be accompanied by **CSV** data checker to be sure the file which will be provided to the API is a **DAG** (Directed Acyclic Graph).

> [!NOTE]
> The dataset provided by the customer is a cleaned version of the dataset from the **[9th DIMACS Implementation Challenge - Shortest Paths](https://www.diag.uniroma1.it/challenge9/download.shtml#benchmark)**
>
> | Datasets                             | Links                                                                                                           |
> | ------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
> | Cleaned *(provided by the customer)* | [DropBox](https://www.dropbox.com/transfer/AAAAAOehAVYlkjxxZRCMqTskDm1obt66RxD2g8zqp0-QqnCZofEMkjw)             |
> | Orignal                              | [www.diag.uniroma1.it/challenge9 *(Instant Download)*](https://www.diag.uniroma1.it/challenge9/data/USA-road-t/USA-road-t.USA.gr.gz) |

## How to Use the Software 🛠️

### Prerequisites

- **Git** (version 2.39.5 or higher)
- **CMake** (version 3.30.3 or higher)
- **CLang** (version 19.1.5 or higher)
- **GoogleTest** (version 1.15.2 or higher) *(Only required for the test suite)*
- **MinGW** (version 12.2 or higher) *(Only required for Windows)*

### Data Checker Installation (DGV)

1. Prepare the project:
   1. Open your terminal and clone the project:

> [!IMPORTANT]
> Git is required for this step.

```bash
git clone https://github.com/algosup/2024-2025-project-3-quickest-path-team-5
```

   2. Navigate to the project directory:

       ```bash
       cd 2024-2025-project-3-quickest-path-team-5
       ```

   3. **\[OPTIONAL\]:** If you want to run the test suite, you need to import **GoogleTest**:

       1. Go to the [GoogleTest Github releases](https://github.com/google/googletest/releases)
       2. Download the version 1.15.2 OR latest.
       3. Extract the downloaded file.
       4. Copy the `googletest-main` folder to the `2024-2025-project-3-quickest-path-team-5` directory.
       5. Rename the folder `googletest-main` to `googletest`.

   4. Navigate to the dag graph validator directory:

       ```bash
       cd dag_graph_validator
       ```

2. Compile the project:

   1. Generate Build Files using CMake:

> [!IMPORTANT]
> For **Windows**:
>
> - **MinGW** is required for this step.
>
> ```bash
> cmake -S . -B build -G "MinGW Makefiles"
> ```

<br>

> [!IMPORTANT]
> For **MacOS/Linux**:
>
> ```bash
> cmake -S . -B build
> ```

<br>

> [!WARNING]
> If you want to also build the test suite, verify your terminal:
>
> | Message                                     | Meaning                                               |
> | ------------------------------------------- | ----------------------------------------------------- |
> | GoogleTest not found. Skipping tests build. | You wrongly install GoogleTest. Take look to step 1.3 |
> | GoogleTest found. Building tests.           | Everything is good and the test suite will be build   |

   2. Navigate to the build directory:

       ```bash
       cd build
       ```

   3. Generate executable(s):

> [!IMPORTANT]
> For **Windows**:
>
> - **MinGW** is required for this step.
>
> ```bash
> mingw32-make
> ```

<br>

> [!IMPORTANT]
> For **MacOS/Linux**:
>
> ```bash
> make
> ```

3. Run the program(s):

    1. Navigate to the bin directory where executables are stored:

> [!WARNING]
> If you are in the build folder, you need to go back to the project directory before going to the bin directory:
>
> ```bash
> cd ..
> ```

<br>

```bash
cd ../bin
```

   2. Run the programs:

       - To run the application, execute the following command:

       ```bash
       ./dgv
       ```

       - To run the test suite, execute the following command:

       ```bash
       ./tests
       ```
